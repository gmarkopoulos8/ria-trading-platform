# Replit Agent Prompt — Fully Autonomous Alpaca Paper Trading Bot

This wires the existing scan scheduler, signal pipeline, and Alpaca execution together into a completely hands-off autonomous loop. After this is applied, the bot will run every weekday with zero manual intervention required.

Only touch the files listed. Do not rebuild anything that already exists.

---

## WHAT ALREADY EXISTS — DO NOT REBUILD

- `dailyScanScheduler.ts` → fires `runDailyScan()` at 9:30 AM ET ✅
- `runDailyScan()` → scans universe, persists results, returns `scanRunId` ✅
- `buildSignalsFromLatestScan()` → pulls top signals from DB ✅
- `runTradingCycle()` → executes trades for a user ✅
- `IntradayMonitorLoop` → monitors positions every 5 min ✅
- `closeAllPositions()` → closes all Alpaca paper positions at once ✅
- `sendDailySummary()` → sends Telegram end-of-day report ✅
- `UniversalAdaptiveEngine` → adapts parameters per exchange ✅
- `hasAlpacaCredentials()` → checks if Alpaca is connected ✅

**The only missing piece:** nothing calls `runTradingCycle()` after `runDailyScan()` completes. Everything else is already wired. This prompt adds that connection plus 3 other lifecycle hooks.

## PHASE 2 — Autonomous Executor Service

This is the single function called by all four triggers. It finds the active user, checks Alpaca is connected, runs adaptive parameters, fetches signals, and executes trades. It is idempotent — safe to call multiple times and will not double-trade.

**File**: `server/src/services/autotrader/AutonomousExecutor.ts` (NEW FILE)

```typescript
import { prisma } from '../../lib/prisma';
import { buildSignalsFromLatestScan } from '../scans/dynamicUniverseService';
import { runTradingCycle, type AutoTradeConfig, DEFAULT_AUTO_TRADE_CONFIG } from './AutoTradeExecutor';
import { computeAdaptive, getCachedAdaptive, registerForAdaptation, DEFAULT_BOUNDS } from './UniversalAdaptiveEngine';
import { hasAlpacaCredentials, getAlpacaCredentials } from '../alpaca/alpacaConfig';
import telegramService from '../notifications/TelegramService';

export interface AutonomousRunResult {
  userId:          string;
  scanRunId?:      string;
  signalsFound:    number;
  tradesPlaced:    number;
  tradesRejected:  number;
  dryRun:          boolean;
  adaptiveRegime:  string;
  effectiveConviction: number;
  errors:          string[];
  ranAt:           Date;
}

export async function runAutonomousCycle(
  trigger: 'MARKET_OPEN' | 'SCAN_COMPLETE' | 'REENTRY' | 'MANUAL',
  scanRunId?: string,
): Promise<AutonomousRunResult[]> {
  const results: AutonomousRunResult[] = [];

  // Find all users with autonomous mode enabled
  const users = await prisma.userSettings.findMany({
    where: { autonomousMode: true, autoTradeEnabled: true },
  });

  if (users.length === 0) {
    console.info('[Autonomous] No users with autonomous mode enabled — skipping');
    return results;
  }

  for (const settings of users) {
    const result: AutonomousRunResult = {
      userId:             settings.userId,
      scanRunId,
      signalsFound:       0,
      tradesPlaced:       0,
      tradesRejected:     0,
      dryRun:             true,
      adaptiveRegime:     'UNKNOWN',
      effectiveConviction: settings.autonomousMinConviction ?? 75,
      errors:             [],
      ranAt:              new Date(),
    };

    try {
      // Check Alpaca is connected
      if (!hasAlpacaCredentials()) {
        result.errors.push('Alpaca not connected — skipping autonomous cycle');
        results.push(result);
        continue;
      }

      const creds = getAlpacaCredentials();
      result.dryRun = creds?.dryRun ?? true;

      // Parse user config
      const rawConfig = typeof settings.autoTradeConfig === 'object' && settings.autoTradeConfig !== null
        ? (settings.autoTradeConfig as Partial<AutoTradeConfig>)
        : {};

      const baseConfig: AutoTradeConfig = {
        ...DEFAULT_AUTO_TRADE_CONFIG,
        ...rawConfig,
        enabled:           true,
        exchange:          'PAPER',
        dryRun:            result.dryRun,
        maxOpenPositions:  settings.autonomousMaxPositions ?? 3,
        maxPositionPct:    settings.autonomousCapitalPct ?? 5.0,
        minConvictionScore: settings.autonomousMinConviction ?? 75,
      };

      // Get adaptive parameters
      let adaptiveParams = getCachedAdaptive(settings.userId, 'PAPER');
      if (!adaptiveParams) {
        adaptiveParams = await computeAdaptive(
          settings.userId,
          'PAPER',
          {
            stopLossPct:        baseConfig.stopLossPct,
            takeProfitPct:      baseConfig.takeProfitPct,
            minConvictionScore: baseConfig.minConvictionScore,
          },
          DEFAULT_BOUNDS['PAPER'],
        ).catch(() => null);

        if (adaptiveParams) {
          registerForAdaptation(settings.userId, 'PAPER', {
            stopLossPct:        baseConfig.stopLossPct,
            takeProfitPct:      baseConfig.takeProfitPct,
            minConvictionScore: baseConfig.minConvictionScore,
          });
        }
      }

      result.adaptiveRegime = adaptiveParams?.regime ?? 'UNKNOWN';

      // Apply adaptive overrides
      const effectiveConviction = adaptiveParams
        ? Math.max(baseConfig.minConvictionScore, adaptiveParams.minConvictionScore)
        : baseConfig.minConvictionScore;

      result.effectiveConviction = effectiveConviction;

      if (adaptiveParams) {
        if (adaptiveParams.stopLossPct < baseConfig.stopLossPct) {
          baseConfig.stopLossPct = adaptiveParams.stopLossPct;
        }
        if (adaptiveParams.takeProfitPct > baseConfig.takeProfitPct) {
          baseConfig.takeProfitPct = adaptiveParams.takeProfitPct;
        }
      }

      // Fetch signals from latest completed scan
      const rawSignals = await buildSignalsFromLatestScan({
        minConvictionScore: effectiveConviction,
        minConfidenceScore: 60,
        allowedBiases:      ['BULLISH'],
        maxSymbols:         (settings.autonomousMaxPositions ?? 3) * 5,
      });

      // Re-rank by quality score
      const signals = [...rawSignals]
        .sort((a: any, b: any) => {
          const aScore = (a.thesisHealthScore ?? 0) * 0.4 + (a.convictionScore ?? 0) * 0.4 + (a.confidenceScore ?? 0) * 0.2;
          const bScore = (b.thesisHealthScore ?? 0) * 0.4 + (b.convictionScore ?? 0) * 0.4 + (b.confidenceScore ?? 0) * 0.2;
          return bScore - aScore;
        })
        .slice(0, settings.autonomousMaxPositions ?? 3)
        .map((s: any) => ({ ...s, exchange: 'PAPER' as const }));

      result.signalsFound = signals.length;

      if (signals.length === 0) {
        console.info(`[Autonomous] No qualifying signals for ${settings.userId} (conviction >= ${effectiveConviction}, regime: ${result.adaptiveRegime})`);
        results.push(result);
        continue;
      }

      // Execute trading cycle
      const cycleResults = await runTradingCycle(settings.id, baseConfig, signals);

      result.tradesPlaced   = cycleResults.filter(r => ['FILLED', 'DRY_RUN'].includes(r.status)).length;
      result.tradesRejected = cycleResults.filter(r => !['FILLED', 'DRY_RUN'].includes(r.status)).length;

      // Update last run timestamp
      await prisma.userSettings.update({
        where: { id: settings.id },
        data:  { lastAutonomousRun: new Date() },
      });

      // Send Telegram notification if trades were placed
      if (result.tradesPlaced > 0) {
        const placed = cycleResults.filter(r => ['FILLED', 'DRY_RUN'].includes(r.status));
        const topTrade = placed[0];
        await telegramService.notify({
          type:     'TRADE_PLACED',
          exchange: 'hyperliquid', // reuse existing type, shows as PAPER in message
          ticker:   topTrade?.symbol,
          data: {
            ticker:     placed.map(r => r.symbol).join(', '),
            side:       'LONG',
            quantity:   placed.length,
            entryPrice: topTrade?.entryPrice?.toFixed(2) ?? '?',
            stop:       'adaptive',
            target:     'adaptive',
            conviction: result.effectiveConviction,
            rr:         (baseConfig.takeProfitPct / baseConfig.stopLossPct).toFixed(1),
            dryRun:     result.dryRun,
            regime:     result.adaptiveRegime,
            trigger,
          },
        }).catch(() => { /* Telegram non-fatal */ });
      }

      console.info(`[Autonomous] ${trigger} cycle complete for ${settings.userId}: ${result.tradesPlaced} placed, ${result.tradesRejected} rejected, regime: ${result.adaptiveRegime}`);

    } catch (err: any) {
      result.errors.push(err?.message ?? 'Unknown error');
      console.error('[Autonomous] Cycle error:', err?.message);
    }

    results.push(result);
  }

  return results;
}
```

Add this helper function before `schedulerTick()`:

```typescript
function isEndOfDayTime(date: Date): boolean {
  // 3:45 PM ET — close all positions and send summary
  return date.getHours() === 15 && date.getMinutes() === 45;
}

async function runEndOfDay(): Promise<void> {
  console.log('[Scheduler] End-of-day routine starting');

  // Close all open Alpaca paper positions
  if (hasAlpacaCredentials()) {
    try {
      const { closed, errors } = await closeAllPositions();
      console.log(`[Scheduler] EOD: closed ${closed} positions, ${errors.length} errors`);
      if (errors.length > 0) {
        console.warn('[Scheduler] EOD close errors:', errors);
      }
    } catch (err: any) {
      console.error('[Scheduler] EOD close all failed:', err?.message);
    }
  }

  // Write any open FILLED autoTradeLog entries as CLOSED with EOD reason
  try {
    const today = new Date();
    today.setHours(0, 0, 0, 0);
    await prisma.autoTradeLog.updateMany({
      where: {
        exchange: 'PAPER',
        status:   'FILLED',
        phase:    'ENTRY',
        executedAt: { gte: today },
      },
      data: {
        status:    'CLOSED',
        exitPrice: 0,  // will be overwritten by actual price on next monitor cycle
        reason:    'EOD_FORCED_CLOSE',
      },
    });
  } catch (err: any) {
    console.warn('[Scheduler] EOD log cleanup error:', err?.message);
  }

  // Send Telegram daily summary
  await telegramService.sendDailySummary().catch(() => {});

  console.log('[Scheduler] End-of-day routine complete');
}
```

```typescript
// Enable/configure autonomous mode
router.post('/autonomous/enable', async (req: Request, res: Response) => {
  try {
    const userId = req.session!.userId as string;
    const {
      minConviction = 75,
      maxPositions  = 3,
      capitalPct    = 5.0,
    } = req.body;

    const settings = await getUserSettings(userId);
    await prisma.userSettings.update({
      where: { id: settings.id },
      data: {
        autonomousMode:          true,
        autoTradeEnabled:        true,
        autonomousMinConviction: minConviction,
        autonomousMaxPositions:  maxPositions,
        autonomousCapitalPct:    capitalPct,
      },
    });

    res.json({ success: true, data: { autonomousMode: true, minConviction, maxPositions, capitalPct } });
  } catch (err) {
    res.status(500).json({ success: false, error: err instanceof Error ? err.message : 'Enable failed' });
  }
});

router.post('/autonomous/disable', async (req: Request, res: Response) => {
  try {
    const userId   = req.session!.userId as string;
    const settings = await getUserSettings(userId);
    await prisma.userSettings.update({
      where: { id: settings.id },
      data:  { autonomousMode: false },
    });
    res.json({ success: true, data: { autonomousMode: false } });
  } catch (err) {
    res.status(500).json({ success: false, error: err instanceof Error ? err.message : 'Disable failed' });
  }
});

// Manually trigger one autonomous cycle right now
router.post('/autonomous/run', requireAuth, async (req: Request, res: Response) => {
  try {
    const results = await runAutonomousCycle('MANUAL');
    res.json({ success: true, data: { results, cyclesRun: results.length } });
  } catch (err) {
    res.status(500).json({ success: false, error: err instanceof Error ? err.message : 'Manual run failed' });
  }
});

// Get autonomous mode status
router.get('/autonomous/status', requireAuth, async (req: Request, res: Response) => {
  try {
    const userId   = req.session!.userId as string;
    const settings = await getUserSettings(userId);
    res.json({
      success: true,
```

```tsx
        </div>
      ))}

      <div className="flex items-start gap-2 p-2.5 bg-amber-500/10 border border-amber-500/20 rounded-lg mt-2">
        <AlertTriangle className="h-3.5 w-3.5 text-amber-400 flex-shrink-0 mt-0.5" />
        <p className="text-[11px] text-amber-400">
          Autonomous mode will trade every weekday without manual confirmation.
          {autonomousStatus?.alpacaDryRun
            ? ' Dry Run is ON — orders are simulated, no real paper trades.'
            : ' Dry Run is OFF — real paper trades will execute on Alpaca.'}
          {' '}You can disable at any time or use the Emergency Exit button.
        </p>
      </div>
    </div>
  )}

  {/* Active mode: show daily schedule */}
  {autonomousStatus?.autonomousMode && (
    <div className="mt-3 pt-3 border-t border-emerald-500/20">
      <div className="grid grid-cols-2 md:grid-cols-4 gap-2 text-xs">
        {[
          { time: '9:00 AM', label: 'Pre-market scan',    color: 'text-slate-400' },

          { time: '3:45 PM', label: 'Close all + report', color: 'text-amber-400' },
        ].map(({ time, label, color }) => (
          <div key={time} className="bg-surface-2 rounded-lg p-2 text-center border border-surface-border">
            <p className={cn('font-mono font-bold text-sm', color)}>{time}</p>
            <p className="text-slate-500 text-[10px] mt-0.5">{label}</p>
          </div>
        ))}
      </div>
      <div className="flex items-center gap-4 mt-2 text-xs text-slate-500">
        <span>Conviction: <strong className="text-white">{autonomousStatus.autonomousMinConviction}</strong></span>
        <span>Max positions: <strong className="text-white">{autonomousStatus.autonomousMaxPositions}</strong></span>
        <span>Capital/trade: <strong className="text-white">{autonomousStatus.autonomousCapitalPct}%</strong></span>

</Card>
```

## PHASE 7 — Required Replit Secret

In Replit Secrets, make sure this is set:
```
ENABLE_DAILY_SCANS=true
```

Without this, the scheduler won't fire and the bot won't run automatically. The default value in the scheduler is `true` unless explicitly set to `false`, but setting it explicitly ensures it doesn't accidentally get disabled.

Optionally also set:
```
PREMARKET_SCAN_ENABLED=true
```
This triggers a lighter pre-market scan at 9:00 AM so signals are ready before the 9:30 AM execution cycle.

### The Full Daily Lifecycle Once Enabled

```
8:00 AM   Server starts / already running
9:00 AM   Pre-market scan (optional, if PREMARKET_SCAN_ENABLED=true)
9:30 AM   Market opens → scheduler fires → daily scan runs (5-30 min)
          → scan completes → AutonomousExecutor picks top signals
          → adaptive engine adjusts parameters for current regime
          → trades execute on Alpaca paper
          → Telegram: "2 trades placed: NVDA, AAPL — Bull Trend, conviction 82"

All day   IntradayMonitorLoop runs every 5 min
          → checks stop/target on open positions
          → when position closes → 30s delay → re-entry scan
          → intraday signal engine scans for fast momentum setups

3:45 PM   End-of-day routine runs
          → closeAllPositions() wipes everything on Alpaca
          → autoTradeLog entries marked CLOSED
          → Telegram: daily summary with P&L, trades placed, closed

Next day  Repeats
```

### Nothing to do after enabling

Once you click **Enable** on the AutoTrader page and set your 3 parameters, the bot runs on its own every weekday. You can monitor it via the AutoTrader logs, the Alpaca tab (live positions/equity), and Telegram notifications. The **Run Now** button lets you manually trigger a cycle any time for testing.
