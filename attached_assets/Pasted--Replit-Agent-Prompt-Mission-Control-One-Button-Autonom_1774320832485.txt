# Replit Agent Prompt — Mission Control: One-Button Autonomous Trading

Build the "Mission Control" page — a single screen where the user clicks one button to start fully autonomous trading. Claude picks the stocks, crypto, and options. Everything runs without human intervention. This replaces the AutoTrader page as the primary interface.

Also fix the remaining gaps: crypto routing for Alpaca, missing autonomousDisable route, and the `autoTradeEnabled` field in the autonomous toggle logic.

---

## WHAT'S ALREADY WORKING (do not rebuild)
- `ClaudeTradingBrain.ts` — fully wired, makes all trade decisions ✅
- `AutonomousExecutor.ts` — runs the cycle for all users with autonomousMode=true ✅
- `dailyScanScheduler.ts` — triggers scan + autonomous cycle at 9:30 AM ET ✅
- EOD close at 3:45 PM + Telegram summary at 4 PM ✅
- Alpaca stock execution, stop/target monitor ✅
- Intraday monitor loop (minute-level) ✅
- Regime detector with correct thresholds ✅

## WHAT NEEDS FIXING

### Gap 1 — Crypto symbols fail on Alpaca
Crypto assets (BTC, ETH, SOL, etc.) from the scan go to `PAPER` exchange → Alpaca. Alpaca crypto uses different symbols (`BTC/USD` not `BTC`) and `ioc` not `day` time-in-force. The current code sends `BTC` as a stock order and Alpaca rejects it.

### Gap 2 — `autonomousDisable` route missing from `autotrader.ts`
The `/autonomous/disable` route doesn't exist. The UI disable button would 404.

### Gap 3 — Autonomous enable doesn't set `autoTradeEnabled: true`
Setting `autonomousMode: true` without `autoTradeEnabled: true` means the executor finds no qualifying users. Both fields need to be set together.

### Gap 4 — No unified "GO" interface
The user has to: run a scan, then enable autotrader, then enable autonomous mode, then run a cycle. There's no single button that starts the whole thing.

// Alpaca crypto symbols use / separator and ioc time-in-force
function toAlpacaCryptoSymbol(symbol: string): string {
  const map: Record<string, string> = {
    'BTC': 'BTC/USD', 'ETH': 'ETH/USD', 'SOL': 'SOL/USD', 'DOGE': 'DOGE/USD',
    'AVAX': 'AVAX/USD', 'LINK': 'LINK/USD', 'XRP': 'XRP/USD', 'DOT': 'DOT/USD',
    'ADA': 'ADA/USD', 'MATIC': 'MATIC/USD', 'ATOM': 'ATOM/USD', 'NEAR': 'NEAR/USD',
    'APT': 'APT/USD', 'ARB': 'ARB/USD', 'OP': 'OP/USD', 'INJ': 'INJ/USD',
    'BNB': 'BNB/USD', 'LTC': 'LTC/USD', 'UNI': 'UNI/USD',
  };
  return map[symbol.toUpperCase()] ?? `${symbol.toUpperCase()}/USD`;
}

const CRYPTO_SYMBOLS = new Set([
  'BTC','ETH','SOL','DOGE','AVAX','LINK','XRP','DOT','ADA','MATIC',
  'ATOM','NEAR','APT','ARB','OP','INJ','BNB','LTC','UNI',
]);

export function isCryptoSymbol(symbol: string): boolean {
  return CRYPTO_SYMBOLS.has(symbol.toUpperCase()) || symbol.includes('/');
}
```

// GET — everything Mission Control needs in one request
router.get('/mission-control', async (req: Request, res: Response) => {
  try {
    const userId   = req.session!.userId as string;
    const settings = await getUserSettings(userId);

    // Reload Alpaca credentials if needed
    let alpacaConnected = hasAlpacaCredentials();
    if (!alpacaConnected) {
      try {
        const { credentialService } = await import('../services/credentials/CredentialService');
        const creds = await credentialService.getAlpacaCredentials(userId);
        if (creds) { setAlpacaRuntimeCredentials(creds); alpacaConnected = true; }
      } catch { /* non-fatal */ }
    }

    const [regime, portfolio, latestScan, recentLogs, signals] = await Promise.allSettled([
      import('../services/market/RegimeDetector').then(m => m.detectRegime()),
      import('../services/portfolio/PortfolioStateService').then(m => m.getPortfolioState()),
      prisma.dailyScanRun.findFirst({
        where: { status: 'COMPLETED' },
        orderBy: { completedAt: 'desc' },
        select: { id: true, completedAt: true, resultCount: true, runType: true },
      }),
      prisma.autoTradeLog.findMany({
        where:   { userSettingsId: settings.id },
        orderBy: { executedAt: 'desc' },
        take:    10,
        select:  { symbol: true, status: true, exchange: true, entryPrice: true,
                   exitPrice: true, pnl: true, executedAt: true, reason: true,
                   convictionScore: true, assetClass: true },
      }),
      buildSignalsFromLatestScan({
        minConvictionScore: Math.max(60, ((settings as any).autonomousMinConviction ?? 75) - 10),
        minConfidenceScore: 55,
        allowedBiases:      ['BULLISH', 'BEARISH', 'NEUTRAL'],
        maxSymbols:         8,
      }).catch(() => []),
    ]);

    const todayStart = new Date();
    todayStart.setHours(0, 0, 0, 0);
    const todayStats = await prisma.autoTradeLog.aggregate({
      where: { userSettingsId: settings.id, executedAt: { gte: todayStart } },
      _count: { id: true },
      _sum:   { pnl: true },
    });

    res.json({
      success: true,
      data: {
        // Mode state
        autonomousMode:          (settings as any).autonomousMode          ?? false,
        autoTradeEnabled:        settings.autoTradeEnabled,
        autonomousMinConviction: (settings as any).autonomousMinConviction ?? 75,
        autonomousMaxPositions:  (settings as any).autonomousMaxPositions  ?? 3,
        autonomousCapitalPct:    (settings as any).autonomousCapitalPct    ?? 5.0,
        lastAutonomousRun:       (settings as any).lastAutonomousRun       ?? null,
        dryRun:                  getAlpacaCredentials()?.dryRun ?? true,

        // Connections
        alpacaConnected,

        // Market
        regime:    regime.status === 'fulfilled' ? regime.value : null,
        portfolio: portfolio.status === 'fulfilled' ? portfolio.value : null,

        // Scan
        latestScan: latestScan.status === 'fulfilled' ? latestScan.value : null,

        // Signals
        signals: signals.status === 'fulfilled' ? signals.value : [],

        // Performance
        todayTrades: todayStats._count.id,
        todayPnl:    todayStats._sum.pnl ?? 0,
        recentLogs:  recentLogs.status === 'fulfilled' ? recentLogs.value : [],
      },
    });
  } catch (err) {
    res.status(500).json({ success: false, error: err instanceof Error ? err.message : 'Mission control status failed' });
  }
});
```

          )}
        </Card>

        {/* Recent trades */}
        <Card className="p-4">
          <div className="flex items-center gap-2 mb-3">
            <Clock className="h-4 w-4 text-slate-500" />
            <h3 className="text-sm font-bold text-white">Recent Trades</h3>
            <button onClick={() => refetch()} className="ml-auto text-slate-600 hover:text-slate-400">
              <RefreshCw className="h-3 w-3" />
            </button>
          </div>

          {logs.length === 0 ? (
            <div className="text-center py-8 text-slate-600 text-xs">
              No trades yet — click GO to start
            </div>
          ) : (
            <div className="space-y-1.5">
              {logs.map((log: any, i: number) => {
                const pnl  = log.pnl ?? null;
                const open = !log.exitPrice && log.status !== 'CLOSED';
                return (
                  <div key={i} className={cn(
                    'flex items-center gap-2 text-xs p-2.5 rounded-xl border',
                    pnl !== null && pnl >= 0 ? 'bg-emerald-500/5 border-emerald-500/15' :
                    pnl !== null && pnl < 0  ? 'bg-red-500/5 border-red-500/15' :
                    'bg-surface-2 border-surface-border',
                  )}>
                    <span className={cn('text-[10px] font-bold px-1.5 py-0.5 rounded',
                      log.status === 'FILLED'  ? 'bg-emerald-500/20 text-emerald-400' :
                      log.status === 'DRY_RUN' ? 'bg-violet-500/20 text-violet-400' :
                      log.status === 'CLOSED'  ? 'bg-slate-700 text-slate-400' :
                      'bg-amber-500/20 text-amber-400')}>
                      {log.status}
                    </span>
                    <span className="font-mono font-bold text-white w-14">{log.symbol}</span>
                    {log.entryPrice && <span className="text-slate-400">${log.entryPrice.toFixed(2)}</span>}
                    {pnl !== null && (
                      <span className={cn('font-mono font-bold ml-auto', pnl >= 0 ? 'text-emerald-400' : 'text-red-400')}>
                        {pnl >= 0 ? '+' : ''}{fmtDollar(pnl)}
                      </span>
                    )}
                    {open && <span className="text-violet-400 ml-auto text-[10px]">OPEN</span>}
                    <span className="text-slate-600 text-[10px]">
                      {log.executedAt ? new Date(log.executedAt).toLocaleTimeString([], { hour: '2-digit', minute: '2-digit' }) : ''}
                    </span>
                  </div>
                );
              })}
            </div>
          )}
        </Card>
      </div>

      {/* ── How it works (shown when inactive) ── */}
      {!isActive && (
        <Card className="p-5">
          <h3 className="text-sm font-bold text-white mb-3 flex items-center gap-2">
            <Shield className="h-4 w-4 text-violet-400" /> How autonomous trading works
          </h3>
          <div className="grid grid-cols-1 md:grid-cols-3 gap-4 text-xs">
            {[
              {
                icon: '🔍',

              },

                body: 'Claude reviews all qualifying signals with full market context — regime, VIX, your portfolio concentration — and selects the best trades with adjusted position sizes and stop levels.',
              },

                icon: '⚡',

                body: 'All open positions close before market close every day. No overnight risk.',
              },
              {
                icon: '📱',

              },
              {
                icon: '🛡️',
                title: 'Circuit breakers',
                body: 'Hard stops kick in automatically if daily loss exceeds limit or drawdown hits max. Claude is overridden — safety always wins.',
              },
            ].map(({ icon, title, body }) => (
              <div key={title} className="space-y-1.5">
                <p className="text-base leading-none">{icon}</p>
                <p className="font-semibold text-slate-300">{title}</p>
                <p className="text-slate-500 leading-relaxed">{body}</p>
              </div>
            ))}
          </div>
        </Card>
      )}

      {/* ── Footer status ── */}
      <div className="flex items-center justify-between text-[10px] text-slate-700 pb-4">
        <span>Next scheduled scan: weekdays 9:30 AM ET</span>
        <span>EOD close: 3:45 PM ET · Telegram summary: 4:00 PM ET</span>
        {d?.lastAutonomousRun && (
          <span>Last cycle: {new Date(d.lastAutonomousRun).toLocaleString([], { month: 'short', day: 'numeric', hour: '2-digit', minute: '2-digit' })}</span>
        )}
      </div>
    </div>
  );
}
