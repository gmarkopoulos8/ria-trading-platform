# Replit Agent Prompt — Mission Control: Add Alpaca Equity Panel

The Mission Control page shows total portfolio equity but is missing the Alpaca-specific breakdown — buying power, unrealized P&L on open positions, and a live positions list. This patch adds all of it.

Replace with:

```typescript
const [regime, portfolio, latestScan, recentLogs, signals, alpacaPositions] = await Promise.allSettled([
  import('../services/market/RegimeDetector').then(m => m.detectRegime()),
  import('../services/portfolio/PortfolioStateService').then(m => m.getPortfolioState()),
  prisma.dailyScanRun.findFirst({
    where: { status: 'COMPLETED' },
    orderBy: { completedAt: 'desc' },
    select: { id: true, completedAt: true, resultCount: true, runType: true },
  }),
  prisma.autoTradeLog.findMany({
    where:   { userSettingsId: settings.id },
    orderBy: { executedAt: 'desc' },
    take:    10,
    select:  { symbol: true, status: true, exchange: true, entryPrice: true,
               exitPrice: true, pnl: true, executedAt: true, reason: true,
               convictionScore: true, assetClass: true },
  }),
  buildSignalsFromLatestScan({
    minConvictionScore: Math.max(60, ((settings as any).autonomousMinConviction ?? 75) - 10),
    minConfidenceScore: 55,
    allowedBiases:      ['BULLISH', 'BEARISH', 'NEUTRAL'],
    maxSymbols:         8,
  }).catch(() => []),
  // NEW — fetch live Alpaca positions
  (async () => {
    if (!hasAlpacaCredentials()) return [];
    try {
      const { getPositions, getAccount } = await import('../services/alpaca/alpacaInfoService');
      const [positions, account] = await Promise.all([getPositions(), getAccount()]);
      return {
        positions: (positions ?? []).map((p: any) => ({
          symbol:        p.symbol,
          side:          p.side,
          qty:           parseFloat(p.qty ?? '0'),
          entryPrice:    parseFloat(p.avg_entry_price ?? '0'),
          currentPrice:  parseFloat(p.current_price ?? '0'),
          unrealizedPl:  parseFloat(p.unrealized_pl ?? '0'),
          unrealizedPlPct: parseFloat(p.unrealized_plpc ?? '0') * 100,
          marketValue:   parseFloat(p.market_value ?? '0'),
        })),
        account: {
          equity:       parseFloat(account.equity ?? '0'),
          buyingPower:  parseFloat(account.buying_power ?? '0'),
          cash:         parseFloat(account.cash ?? '0'),
          lastEquity:   parseFloat(account.last_equity ?? account.equity ?? '0'),
        },
      };
    } catch { return { positions: [], account: null }; }
  })(),
]);
```

## WHAT THIS ADDS

The Alpaca panel now shows:

| Field | Source |
|-------|--------|
| **Account Equity** | Real Alpaca account `equity` field — updates live |
| **Day P&L** | `equity - last_equity` — how much made/lost vs yesterday's close |
| **Buying Power** | Real Alpaca `buying_power` — exact cash available to deploy |
| **Open Positions count** | Live from `/v2/positions` |
| **Unrealized P&L** | Sum of all open position `unrealized_pl` |
| **Per-position table** | Symbol, side, qty, entry price, current price, market value, % P&L, $ P&L |

All data refreshes every 15 seconds (same as the rest of the page). Color-coded green/red on every P&L field.
