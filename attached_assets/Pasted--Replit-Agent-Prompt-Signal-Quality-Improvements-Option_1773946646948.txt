## PHASE 1 — Fix Relative Strength (Real Benchmark Comparison)

The current `computeRelativeStrength()` in `server/src/services/technical/indicators/relativeStrength.ts` compares a ticker against itself — it measures absolute momentum, not relative strength. Replace the entire implementation.

**New implementation:**

```typescript
import { OHLCVBar, RelativeStrengthResult, SignalDirection } from '../types';

// SPY benchmark bars are passed in alongside the ticker bars
// This is the only correct way to compute relative strength
export function computeRelativeStrength(
  tickerBars: OHLCVBar[],
  benchmarkBars: OHLCVBar[] | null  // SPY for stocks, BTC for crypto
): RelativeStrengthResult {
  if (tickerBars.length < 20) {
    return { value: 50, percentile: 50, signal: 'NEUTRAL', explanation: 'Insufficient data for relative strength.' };
  }

  // If no benchmark provided, fall back to absolute momentum (old behavior)
  if (!benchmarkBars || benchmarkBars.length < 20) {
    return computeAbsoluteMomentum(tickerBars);
  }

  // Align the two series by matching the most recent N bars
  const n = Math.min(tickerBars.length, benchmarkBars.length, 60); // use last 60 bars
  const tBars = tickerBars.slice(-n);
  const bBars = benchmarkBars.slice(-n);

  // Compute cumulative returns for both
  const tStart = tBars[0].close;
  const bStart = bBars[0].close;
  const tEnd = tBars[tBars.length - 1].close;
  const bEnd = bBars[bBars.length - 1].close;

  const tReturn = (tEnd - tStart) / tStart;
  const bReturn = (bEnd - bStart) / bStart;
  const relativeReturn = tReturn - bReturn; // positive = outperforming benchmark

  // Also compute 20-bar rolling relative strength
  const rollingRS: number[] = [];
  for (let i = 20; i <= n; i++) {
    const tSlice = tBars.slice(i - 20, i);
    const bSlice = bBars.slice(i - 20, i);
    const tr = (tSlice[tSlice.length - 1].close - tSlice[0].close) / tSlice[0].close;
    const br = (bSlice[bSlice.length - 1].close - bSlice[0].close) / bSlice[0].close;
    rollingRS.push(tr - br);
  }

  // Score 0–100: 50 = matches benchmark, >50 = outperforming, <50 = underperforming
  // Map relativeReturn to 0-100 scale: +20% outperformance = 100, -20% = 0
  const value = Math.min(99, Math.max(1, Math.round(50 + relativeReturn * 250)));
  const percentile = value;

  let signal: SignalDirection = 'NEUTRAL';
  if (value >= 65) signal = 'BULLISH';
  else if (value <= 35) signal = 'BEARISH';

  // Check if trend is improving or deteriorating
  const recentRS = rollingRS.slice(-5);
  const olderRS = rollingRS.slice(-10, -5);
  const recentAvg = recentRS.length > 0 ? recentRS.reduce((a, b) => a + b, 0) / recentRS.length : 0;
  const olderAvg = olderRS.length > 0 ? olderRS.reduce((a, b) => a + b, 0) / olderRS.length : 0;
  const trend = recentAvg > olderAvg ? 'improving' : recentAvg < olderAvg ? 'deteriorating' : 'stable';

  const direction = relativeReturn >= 0 ? 'outperforming' : 'underperforming';
  const pct = Math.round(Math.abs(relativeReturn) * 1000) / 10;

  const explanation = `Relative strength ${value}/100 — ${direction} benchmark by ${pct}% over ${n} bars. RS trend is ${trend}.`;

  return { value, percentile, signal, explanation };
}

function computeAbsoluteMomentum(bars: OHLCVBar[]): RelativeStrengthResult {
  // Fallback when no benchmark — absolute win rate method (unchanged from original)
  const n = Math.min(bars.length, 60);
  const recent = bars.slice(-n);
  const step = Math.max(1, Math.floor(n / 20));
  const windowReturns: number[] = [];
  for (let i = step; i < recent.length; i += step) {
    windowReturns.push((recent[i].close - recent[i - step].close) / recent[i - step].close);
  }
  const positiveWindows = windowReturns.filter((r) => r > 0).length;
  const winRate = windowReturns.length > 0 ? positiveWindows / windowReturns.length : 0.5;
  const value = Math.min(99, Math.max(1, Math.round(winRate * 100)));
  let signal: SignalDirection = 'NEUTRAL';
  if (value >= 65) signal = 'BULLISH';
  else if (value <= 35) signal = 'BEARISH';
  return {
    value, percentile: value, signal,
    explanation: `Absolute momentum ${value}/100 (no benchmark available). Positive in ${Math.round(winRate * 100)}% of windows.`,
  };
}
```

// Hyperliquid: POST /info with { type: 'metaAndAssetCtxs' } returns funding rates
export async function getFundingRate(asset: string): Promise<{ rate: number; annualized: number } | null> {
  try {
    const data = await postInfo<[UniverseMeta, AssetContext[]]>({ type: 'metaAndAssetCtxs' });
    const [meta, contexts] = data;
    const idx = meta.universe.findIndex(a => a.name.toUpperCase() === asset.toUpperCase());
    if (idx === -1) return null;
    const ctx = contexts[idx];
    const rate = parseFloat(ctx?.funding ?? '0');
    return { rate, annualized: rate * 3 * 365 * 100 }; // 3 funding periods/day × 365 days × 100 for pct
  } catch {
    return null;
  }
}

export async function getInsiderSignal(ticker: string): Promise<{
  recentBuys: number;
  recentSells: number;
  netSharesBought: number;
  signal: 'BULLISH' | 'BEARISH' | 'NEUTRAL';
  explanation: string;
} | null> {
  try {
    await finnhubRateLimiter.waitForToken();
    const from = new Date(Date.now() - 90 * 86400000).toISOString().slice(0, 10); // last 90 days
    const to = new Date().toISOString().slice(0, 10);
    const { data } = await axios.get(
      `${FINNHUB_BASE}/stock/insider-transactions?symbol=${ticker}&from=${from}&to=${to}&token=${API_KEY}`
    );

    const transactions: Array<{ name: string; share: number; change: number; transactionCode: string; transactionPrice: number }> =
      data?.data ?? [];

    // Only care about open market buys (P) and sells (S) — not grants or exercises
    const buys = transactions.filter(t => t.transactionCode === 'P' && t.change > 0);
    const sells = transactions.filter(t => t.transactionCode === 'S' && t.change < 0);

    const netShares = buys.reduce((s, t) => s + t.change, 0) + sells.reduce((s, t) => s + t.change, 0);
    const buyCount = buys.length;
    const sellCount = sells.length;

    let signal: 'BULLISH' | 'BEARISH' | 'NEUTRAL' = 'NEUTRAL';
    let explanation = '';

    if (buyCount >= 2 && netShares > 0) {
      signal = 'BULLISH';
      explanation = `${buyCount} insider purchase${buyCount > 1 ? 's' : ''} in last 90 days — net ${netShares.toLocaleString()} shares bought. Strong insider conviction.`;
    } else if (sellCount >= 3 && netShares < 0) {
      signal = 'BEARISH';
      explanation = `${sellCount} insider sales in last 90 days. Distribution by insiders.`;
    } else {
      explanation = `${buyCount} buys, ${sellCount} sells in last 90 days. No clear insider signal.`;
    }

    return { recentBuys: buyCount, recentSells: sellCount, netSharesBought: netShares, signal, explanation };
  } catch {
    return null;
  }
}

**`getRecommendation(thesis: FullThesisResult, accountEquity: number, maxRiskDollars: number): Promise<OptionsRecommendation | null>`**:

```typescript
async getRecommendation(thesis, accountEquity, maxRiskDollars) {
  const ticker = thesis.ticker;
  const bias = thesis.thesis.bias;
  const conviction = thesis.thesis.convictionScore;
  const holdWindow = thesis.thesis.suggestedHoldWindow;
  const currentPrice = thesis.marketStructure.currentPrice;

  // Only recommend options for stocks (not crypto — different mechanics)
  if (ticker.length > 5 || currentPrice < 5) return null;

  const [chain, ivRank] = await Promise.all([
    optionsDataService.getOptionsChain(ticker),
    optionsDataService.computeIVRank(ticker),
  ]);

  if (!chain || chain.calls.length === 0) return null;

  const ivR = ivRank?.rank ?? 50;
  const targetDTEMin = 25;
  const targetDTEMax = 50;
  const strategy = selectStrategy(bias, conviction, ivR, holdWindow);

  if (strategy === 'NONE') return null;

  // Build legs based on strategy
  switch (strategy) {
    case 'LONG_CALL':
      return buildLongCall(ticker, chain, currentPrice, ivR, maxRiskDollars, thesis);

    case 'LONG_PUT':
      return buildLongPut(ticker, chain, currentPrice, ivR, maxRiskDollars, thesis);

    case 'BULL_CALL_SPREAD':
      return buildBullCallSpread(ticker, chain, currentPrice, ivR, maxRiskDollars, thesis);

    case 'BEAR_PUT_SPREAD':
      return buildBearPutSpread(ticker, chain, currentPrice, ivR, maxRiskDollars, thesis);

    case 'CASH_SECURED_PUT':
      return buildCashSecuredPut(ticker, chain, currentPrice, ivR, maxRiskDollars, thesis);

    default:
      return null;
  }
}
```

**`selectStrategy()`**:

```typescript
function selectStrategy(bias, conviction, ivRank, holdWindow): OptionsStrategy {
  const holdDays = parseHoldWindowDays(holdWindow); // '2-4 WEEKS' → 21, '1-3 MONTHS' → 60, etc.

  if (bias === 'NEUTRAL') return 'NONE';
  if (conviction < 70) return 'NONE'; // options only on high conviction

  if (bias === 'BULLISH') {
    if (ivRank > 55 && conviction >= 75) return 'CASH_SECURED_PUT'; // sell premium in high IV
    if (conviction >= 82 && holdDays <= 30) return 'LONG_CALL';     // directional bet
    if (conviction >= 70) return 'BULL_CALL_SPREAD';                 // defined risk
  }

  if (bias === 'BEARISH') {
    if (conviction >= 82 && holdDays <= 30) return 'LONG_PUT';
    if (conviction >= 70) return 'BEAR_PUT_SPREAD';
  }

  return 'NONE';
}
```

**Leg selection functions:**

**`buildLongCall()`**:

1. Filter calls to `targetDTEMin–targetDTEMax` range
1. Find contract where `delta` is closest to 0.60 (slightly ITM)
1. Check liquidity: volume ≥ 100, OI ≥ 500, spread ≤ 15% of mid
1. If no liquid contract: return null
1. Number of contracts: `Math.floor(maxRiskDollars / (contract.mid * 100))`; minimum 1
1. Max risk = contracts × mid × 100 (premium paid)
1. Breakeven = strike + mid
1. Probability of profit = approximately `(delta - 0.10) * 100` (rough estimate from delta)

**`buildBullCallSpread()`**:

1. Buy the 0.55–0.65 delta call (25–50 DTE)
1. Sell the call at strike approximately 5% above current price (same expiration)
1. Net debit = long mid - short mid
1. Max risk = net debit × contracts × 100
1. Max profit = (short strike - long strike - net debit) × contracts × 100
1. Ensure reward:risk ≥ 1.5:1; if not, skip
1. Breakeven = long strike + net debit

**`buildCashSecuredPut()`**:

1. Filter puts to 25–35 DTE
1. Find put at or just below nearest support level from thesis (use `thesis.thesis.invalidationZone.level` as strike reference)
1. Target delta of -0.30 to -0.40 (out of the money)
1. Check: stock price must be at least 5% above strike (don’t sell puts too close to current price)
1. Net credit = mid × contracts × 100 (you receive this immediately)
1. Max risk = (strike - mid) × contracts × 100 (if stock goes to zero — very unlikely)
1. Breakeven = strike - mid
1. Cash required = strike × 100 × contracts (must have this in account)

**`buildLongPut()`** and **`buildBearPutSpread()`** — mirror images of LONG_CALL and BULL_CALL_SPREAD respectively.

**`parseHoldWindowDays()`**:

```typescript
function parseHoldWindowDays(w: string): number {
  if (w.includes('INTRADAY')) return 1;
  if (w.includes('1-3 DAYS')) return 2;
  if (w.includes('1-2 WEEKS')) return 10;
  if (w.includes('2-4 WEEKS')) return 21;
  if (w.includes('1-3 MONTHS')) return 60;
  if (w.includes('3-6 MONTHS')) return 120;
  return 21; // default
}
```
